# Changelog

All notable changes to this project will be documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/)
and adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## v0.2.0 — 2026-04-30

### Added

- Introduced new `/news/` section for RSS-powered news pages.
- Added first live page: `/news/world.html`.
- Implemented multi-feed RSS loader using `/js/rss-loader.js`.
- Added global stylesheet `/css/style.css` with full layout, navigation, article cards, and responsive design.
- Added navigation bar linking all core news categories.
- Added bottom fixed ad container for monetisation readiness.

### Changed

- Updated project structure to support multi-page expansion.
- Improved layout consistency across future pages.

### Notes

- This is the first deployment of the RSS news system.
- Additional categories (UK, Tech, Sports, Entertainment) will be cloned from this template after review.
